## How to start an app in react
Create the app
   ````bash
   npx create-react-app [Nombre]
   ````

## After you created the app you can urn several commands:

### Starts the development server.
````bash
  npm start
````
### Bundles the app into static files for production.
````bash
  npm run build
````
### Starts the test runner.
````bash
  npm test
````
### Removes this tool and copies buld dependecies, configuration files and scripts into the app directory. If you do this, you can't go back!.
````bash
  npm run eject
````
